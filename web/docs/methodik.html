Methodik

Elemente (auch sog. Components) sind alle visuellen Dinge auf Webseiten. Siehe Methodik von Brad Frost “Atomic Design”: https://atomicdesign.bradfrost.com/chapter-2/#the-atomic-design-methodology

  • Atome (globale Basiselemente)
    • Abstrakt

      • Schriftart(en)
      • Farben
    • Konkret
      • Überschriften
      • Text-Varianten
  • Moleküle (Kombination von Atomen)
    • Buttons
    • Logo
    • Sitename
    • Slogan
  • Organismen (Kombinierte Moleküle)
    • CTAs
    • Formulare
    • Branding (Logo, Sitename, Slogan)
    • Header
    • Footer
    • Produkt-Grid
  • Templates (Generische Anordnung, Layout)
    • Startseite Layout Varianten
    • Landingpage Layout Varianten
  • Pages (Konkrete Seiten)

    • Landingpage Variante #1
    • etc.
